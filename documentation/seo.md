# Search Visibility

ChemVault Molecule Studio publishes canonical metadata for `https://model.chemvault.science`, Open Graph and summary-card previews, a public sitemap, and crawler rules.

The home page and molecular workspace are indexable product surfaces. Authentication, account library, settings, profile, and API routes are excluded because they are user-specific or machine-facing. The sitemap therefore contains only `/` and `/molecule`.

Preview imagery uses the public ChemVault Molecule Studio mark. Page titles inherit the product title template, while route-specific descriptions should state visible user capabilities rather than implementation details.
